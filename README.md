### 🧠 What These Notes Cover

Fundamentals
Basic networking concepts: IP, MAC, packets, ports, protocols, LAN/WAN, NAT, DHCP.

OSI Model
The 7‑layer model explained simply with real DevOps examples.

DNS
A → IP mapping, CNAME, MX, TXT, resolvers, propagation, dig, nslookup.

Routing
Routing tables, gateways, traceroute, static vs dynamic routing, BGP, OSPF.

Subnetting
CIDR, VPC subnetting, IP ranges, AWS public/private subnets, NAT.

Troubleshooting
Ping, dig, traceroute, curl, netstat, nc — real debugging workflow.

Commands
All useful Linux + networking commands in one place.
